Birthday-Wish 

This is a starter website I created with Vite to experiment and play around with React, JSX, TypeScript, etc. The application displays 'Happy Birthday' balloons as a fun birthday wish. 

User Flow:

1) enter your name in the input
2) click on the prompt button to see the happy birthday appear
3) use the pause button to stop the balloon drop effect loop
4) use the reset button to bring you back to the first page with an empty name input



In this project, some extra features I used: 

- Anime.js to animate the balloon components to move around
